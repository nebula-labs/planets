# Planet building

This repo is a script that simulates two arbitrary chains with:
* IBC capability
* ICA capability
* Wasm capability

It will setup ibc bridge between two chains

# Requirements

1. install jq
2. install docker
3. install docker compose
2. install go 'wget -q -O - https://git.io/vQhTU | bash -s -- --version 1.20.1'
